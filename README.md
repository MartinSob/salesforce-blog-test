## Libraries used
- Angular: Used to diagram the components and how to interact with each other.
- Bootstrap: For styling, Bootstrap was chosen as is the one most programmers are familiar with.
- Axios: To make every request to the API.

## Notes
- To run the application, you first need to run:
```
npm i
npm start
```

## Source code files
- `src/app/blog-detail`: Component for showing the detail of a specific blog
- `src/app/blog-edit`: Component for editing the values of a blog
- `src/app/blog-list`: Component for showing the complete list of blogs and contains the detail of the blog
- `src/app/service`: Is where the service in charge of making every request is
- `src/app/models`: Where every model used in the project is. In this case only the Blog model was necessary.
